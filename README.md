# 🛡️ TruthShield AI - أقوى كاشف كذب للأخبار

مرحباً بك في **TruthShield AI**، المنصة الاحترافية المدعومة بالذكاء الاصطناعي للتحقق من صحة الأخبار والمنشورات. تم بناء هذا المشروع ليكون سريعاً، قابلاً للتوسع، وسهل الاستخدام لجذب ملايين المستخدمين.

## 🚀 المميزات الرئيسية
- **تحليل فوري:** تحقق من أي نص أو رابط في أقل من 3 ثوانٍ.
- **دعم لغات متعددة:** واجهة كاملة بالعربية والإنجليزية مع دعم RTL.
- **تكامل مع OpenRouter:** الوصول إلى أقوى نماذج الذكاء الاصطناعي (Claude, GPT-4, DeepSeek, Llama).
- **PWA Ready:** قابل للتثبيت على الهواتف ويعمل في وضع عدم الاتصال (Offline Demo).
- **تصميم عصري:** واجهة مستخدم مريحة مع دعم الوضع الليلي (Dark Mode).
- **خصوصية تامة:** يتم تخزين مفاتيح API وسجل العمليات محلياً في متصفحك.

## 🛠️ التقنيات المستخدمة
- **HTML5 & CSS3** (Flexbox/Grid)
- **Tailwind CSS v3.4** (عبر CDN)
- **Alpine.js v3** (لإدارة الحالة والمنطق)
- **Lucide Icons** (أيقونات احترافية)
- **OpenRouter API** (محرك الذكاء الاصطناعي)

## 📦 خطوات التشغيل والنشر

### 1. التشغيل المحلي
لا تحتاج إلى تثبيت أي مكتبات! فقط افتح ملف `index.html` في متصفحك.

### 2. إعداد مفتاح API
1. اذهب إلى [OpenRouter.ai](https://openrouter.ai/) وأنشئ حساباً.
2. قم بتوليد مفتاح API جديد.
3. افتح موقع **TruthShield AI**، اضغط على أيقونة المفتاح في الأعلى، والصق المفتاح هناك.

### 3. النشر (Deployment)
المشروع جاهز للنشر الفوري على أي منصة استضافة ثابتة (Static Hosting):
- **Vercel / Netlify:** اسحب المجلد وأفلته (Drag & Drop).
- **GitHub Pages:** ارفع الملفات إلى مستودع (Repository) وقم بتفعيل GitHub Pages من الإعدادات.

## 📂 هيكل الملفات
- `index.html`: الهيكل الأساسي للواجهة.
- `style.css`: التنسيقات المخصصة والرسوم المتحركة.
- `script.js`: منطق التطبيق والتكامل مع API.
- `manifest.json`: إعدادات تطبيق الويب التقدمي (PWA).
- `sw.js`: ملف الـ Service Worker للعمل أوفلاين.

## 🛡️ الأمان والخصوصية
- لا يتم إرسال مفتاح API الخاص بك إلى أي خادم خارجي سوى OpenRouter.
- يتم حفظ السجل في `localStorage` الخاص بمتصفحك فقط.
